##ICO VLB Tokens Contracts

This repository contains Solidity smart contracts, deploy manifests and tests for VLB Tokens ICO campaign.

### How do I get set up? ###
Console #1
```
$: npm install -g yarn
$: yarn install
```

Console #2
```
$: ./run-testrpc.sh
```

Console #1
```
$: npm run compile
$: npm run migrate
$: npm test
```
